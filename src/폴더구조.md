# FE 최종 폴더구조

## 요약

/src
├── 📱 app # [Routing] Next.js App Router 기반의 페이지 및 경로 관리
├── 🔌 apis # [API] 서버 통신을 위한 API Fetch 함수 모음
├── 🧪 features # [Domain] 비즈니스 로직 + 도메인이 결합된 복합 컴포넌트 (ex. Editor)
├── 🎨 shared # [Global] 도메인에 종속되지 않는 전역 공통 요소
│ ├── components # 공통 UI (Button, Input, Modal 등)
│ ├── hooks # 전역 Custom Hooks
│ ├── constants # [Constant] Query Keys, 에러 메시지 등 고정 상수
│ ├── store # 전역 상태 관리 (Zustand)
│ ├── lib # 라이브러리 설정 (QueryClient, FetchClient)
│ ├── 🛠️ utils # [Utility] 순수 자바스크립트 계산 함수 (Date, Storage 등)
│ └── schemas # 전역 데이터 검증 (Zod)
│
├── 🛡️ proxy.js # [Guard] 권한 가드 및 접근 제한 로직
├── 💅 styles # [Style] 전역 스타일링 및 디자인 시스템
└── 📦 mock # [Mock] 개발용 목데이터
